Multiclass Classification of Dry Beans — #MakeoverMonday

12 April 2021 ● Gandes Goldestan

The story behind the viz for #MakeoverMonday Week 14 submission and got selected as my 1st Favorite Viz of the Week

● ● ●

The 14th dataset for #MakeoverMonday 2021 was out already, and challenge of this week is making a better viz about ‘Multiclass Classification of Dry Beans’.

Original chart

From the scientific paper source we can see that the writer used multiple bar chart with some colors to differentiate the variety of the dry beans. Let’s have a look at the original visualization:

What works well

  • The colour legend is in a prominent place and easy to understand (at least for me).
  • Multiple bar chart style is good enough to explain the rate of classification.
  • The white background makes the design simple and the bar chart is shown clearly.

What could be improved

  • There is no title to explain what chart it is about.
  • The algorithm abbreviation would be useless for those who don’t understand about machine learning if they don’t read the full paper or only see for first sight.
  • Using different color for the label of number would be useful for audience to quickly determine which algorithm has the highest classification rate.
  • Trying to use the bean class as parameter and explain the features should be interesting to learn the characteristic of each beans.

Some people in community said that 14th week dataset is really challenging through their tweets, I definitely agree. Because I was too confused, I decided to watch Andy Kriebel’s #WatchMeViz in his Youtube to get some inspiration before jumping into analysis. You can check for the video here.

What I did

There were so many box plot, scatter plot, and even spider chart I saw from people’s submission to visualize data this week. But after watching Andy’s #WatchMeViz video, I decided to take inspiration from his chart which is focused on relative value of features from each beans. For the chart I wanted to try to use coxcomb style, as I want to learn something new also. Maybe it is a little bit different from other people’s submission, but that’s totally okay for me. Here is the simple bar chart to show relative value (size and %) of the features from each bean class like Andy has done.

To see the percentage of relative value for each class, first thing first I have to make the ‘Relative Value’ calculation. And this is the example of its display as a table form.

Because making a coxcomb chart needs another chart template as data source, I had to make it first in Excel like below:

After I try for a few hours, but I was still not be able to set my chart similarly like a proper coxcomb. It still forms like a propeller. I let it be for now since the result is almost similar like my first intention. It is possible that I will update it later after I find the solution :CRYYYY:

By the way if you want to try coxcomb chart, please check the tutorial from Tableau Magic here to see further.

Next for final process, I added a parameter as feature filter.

Okay since the chart and parameter has done, the final step is designing dashboard and extracting the data. I want to make the style so simple and clean. So I only need to make a title, a brief summary, add a legend, and a calm color for the background (I think a light blue is a cool one).

Finally, the result has appeared!

Check out below for interactive dashboard version! Please hover over the ‘colorful propeller’ to see more details.

Well, as beginner personally I am satisfied enough with the result eventhough I wasn’t be able to make it like a proper coxcomb chart in my inspiration resources. At least, the message I want to deliver to audience is sent by seeing the size and color of each bean class and of course, its tooltip!

Lesson learned

Since this week challenge is a little bit difficult for me personally, I’ve tried hard to look for something different with the chart type. This quote is probably the best one to descript what I’ve experienced this week.

"Dare to be different!"

Yup, this week challenge has given me time to reflect and encourage myself that it is okay to try different chart. Everyone has different perspective, so if you feel that it is best to explain the story, just implement that!

Okay, that’s all story from today. I am ready to prepare for the next challenge. Hopefully this will give you an insight too!
